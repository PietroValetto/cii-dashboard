# Commissioner Influence Index — LinkedIn Edition

A **Kairos Media Partners** index of how the 27 members of the European Commission's College reach and resonate with citizens on **LinkedIn**. It measures **communication performance, not policy quality or political importance.**

**Live dashboard:** https://pietrovaletto.github.io/cii-dashboard/

This repository contains only the public dashboard (a single self-contained `index.html` + profile images). Methodology, data pipeline and analysis live in a separate private repository.
